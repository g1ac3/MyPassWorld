# MyPassWorld
パスワード管理アプリ<br>
PWAUTH.cpp : 
PWAUTH.h : ヘッダファイル
all.txt : パスワード情報等を保存するファイル
mypass.cpp : main関数を含むアプリ本体
pw.txt : 本アプリを使用するためのパスワードを保存するファイル

使用方法
g++ mypass.cpp PWAUTH.cpp -o MyPassWorld

------更新------<br>
2021/06/26 パスワード入力にPWAUTH(https://github.com/g1ac3/PWAUTH) を利用<br>
パスワード初期値"pass"
